Deep Packets Inspection — Real-Time Network Traffic Analyzer

A real-time network traffic monitoring and security analysis platform that captures live packets and flags anomalous behavior.

Tech Stack: MongoDB, Express.js, React.js, Node.js, Scapy, Socket.IO, Chart.js

Key Features:


Live Packet Capture — Uses Scapy to capture and identify protocols including TCP, UDP, HTTP, and HTTPS
Real-Time Dashboard — React + Chart.js visualization of live traffic patterns
Instant Alerts — Socket.IO-powered WebSocket connections deliver real-time security alerts
Threat Detection Engine — Rule-based engine analyzing packet metadata stored in MongoDB to flag anomalous network behavior


Project Structure

deep-packet-inspection/
├── capture/            # Scapy packet-sniffing scripts
├── server/              # Express + Socket.IO backend
├── client/               # React dashboard (Chart.js visualizations)
├── models/             # MongoDB schemas for packets & alerts
└── detection/          # Rule-based threat detection logic

Getting Started

bashgit clone https://github.com/<your-username>/deep-packet-inspection.git
cd deep-packet-inspection
pip install scapy
npm install
npm start

Note: Requires administrator/root privileges to capture live network packets.
